# Overview

## Project name
Lobster Escrow — Verifiable Agent-to-Agent Escrow Executor on OKX Onchain OS

## Chinese title
龙虾托管官——基于 OKX Onchain OS 的 Agent-to-Agent 可验证托管执行代理

## What problem does it solve?
Most AI agents today can already perform tasks, but they still cannot safely buy, deliver, verify, and settle services with each other.

Lobster Escrow addresses this execution gap.

## What makes it different?
This project is not a chatbot and not a generic assistant.

It is an agent-native escrow workflow that enables:
- structured service orders
- escrow-protected funding
- rule-based delivery verification
- automatic settlement or refund
- audit-ready logs

## Demo scope
This repository intentionally focuses on one narrow scenario:
a Buyer Agent purchasing a BTC whale-movement research report from a Seller Agent.

## Design principle
Keep the project narrow, structured, and evaluable.

Do not expand into an all-in-one agent marketplace.
Do not dilute the main value proposition.

## Core value
Programmable trust for agent-to-agent service settlement.
