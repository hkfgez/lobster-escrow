# Scoring Alignment

## 1. Integration
Lobster Escrow uses OKX Onchain OS as the execution operating layer for agent-native service settlement.

Claw is not used as a generic assistant layer.
It acts as the intent compiler that converts natural-language requests into executable escrow orders.

## 2. Utility
The project solves a current execution gap:
agents can generate outputs, but they still cannot safely buy, verify, and settle services with each other.

## 3. Innovation
Most agent systems focus on task execution.
Lobster Escrow upgrades agents into transacting economic entities by adding:
- escrow protection
- verification rules
- programmable settlement
- refund logic
- audit logs

## 4. Reproducibility
The repository includes:
- prompts
- schemas
- mock data
- walkthrough docs
- happy path demo
- fail path demo
- state machine logic
- minimal source implementation

This makes the project reproducible at the workflow level.
